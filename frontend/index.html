Конструктор ссылок

Используйте переменные для формирования ссылки
Доступные переменные:
  • {file} - путь к папке (punycode, со слешами, без имени файла)
  • {fullpath} - полный путь к файлу (punycode, с именем и слешами)
  • {filename} - только имя файла с расширением (punycode)
  • {filename_without_ext} - имя файла без расширения (punycode)
  • {ext} - расширение файла (с точкой, ASCII)
  • {decoded_filename} - имя файла в человекочитаемом виде (UTF-8)
  • {decoded_filename_without_ext} - имя файла без расширения (UTF-8)
  • {bucket} - имя бакета
  • {endpoint} - эндпоинт
  • {dirname} - путь к папке (punycode, со слешем в конце)

Важно: Все ссылки генерируются в punycode для корректной работы с нелатинскими символами.

Пример: https://{bucket}.s3.amazonaws.com/{file}{filename}
