**S3 Manager** — это мощный и удобный веб-интерфейс для управления S3-совместимыми хранилищами. Запускается в Docker и предоставляет интуитивно понятный интерфейс для работы с файлами в облачных хранилищах.

## ✨ Возможности

### 📁 Управление файлами
- **Просмотр содержимого** S3-бакетов с навигацией по папкам
- **Создание папок** прямо в интерфейсе
- **Загрузка файлов** через drag-and-drop или диалоговое окно
- **Рекурсивная загрузка папок** с сохранением структуры
- **Скачивание файлов** с отображением прогресса
- **Удаление файлов и папок** с подтверждением
- **Перемещение файлов** drag-and-drop внутри S3

### 🔗 Конструктор ссылок
- **Генерация публичных ссылок** на файлы по шаблону
- **Punycode поддержка** для нелатинских символов в URL
- **Множественный выбор** файлов для генерации ссылок
- **Экспорт в CSV** с разделителем ";" (UTF-8)
- **Предпросмотр** сгенерированных ссылок

### 🔒 Безопасность
- **Профили подключений** с сохранением эндпоинта и бакета
- **Ключи доступа не сохраняются** — вводятся при каждом подключении
- **SSL/TLS** для всех соединений с S3
- **Интеграция с браузерным менеджером паролей**

### 📊 Мониторинг
- **Прогресс-бары** для загрузки и скачивания
- **Отображение скорости** передачи данных
- **Очередь операций** с счетчиком
- **Вкладка с логами** для отладки
- **Поддержка больших файлов** до 10GB с multipart upload

## 🚀 Быстрый старт

### Предварительные требования
- Docker (версия 20.10 или выше)
- Docker Compose (версия 2.0 или выше)

### Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/yourusername/s3-manager.git
cd s3-manager


2. **Запускаем контейнер:**
```bash
docker compose up -d --build

3. **Открываем приложение в любом браузере**
http://localhost:8080/
